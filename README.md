# Bloomly Women Wellness Web App

`bloomly-frontend/` is the active application.

The old root-level prototype is no longer the source of truth:
- Active app source: `bloomly-frontend/src`
- Active frontend package: `bloomly-frontend/package.json`

Common commands from the repo root:

```bash
npm start
npm run build
npm test
```

Or directly inside the frontend:

```bash
cd bloomly-frontend
npm start
```
